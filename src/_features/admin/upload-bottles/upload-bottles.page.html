Charger votre cave

Pour charger votre cave, connectez-vous sur cavus-vinifera, et aller dans la page exporter vos données sos excel puis exportez par lots. Le fichier XLS sera téléchargé, il ne reste plus qu'à cliquer sur le bouton de chargement, choisir le fichier et votre stock de bouteilles sera chargé et sauvegardé dans le logiciel.

Par défaut:

  • les XLS sont parsés en ascii, le séparateur est la tabulation et les données sont entre guillemets
  • les CSV sont parsés en UTF-8, le séparateur est le point-virgule (pas de guillemets)

    • si les caractères accentués ne passent pas il s'agit d'un problème d'encodage (voir options avancées)
